== Description ==

This plugin allows you to set writing guide for all your users.

== Installation ==

1. Extract the zip file into wp-content/plugins directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= Where are the settings?

In Settings -> Writing Guide
